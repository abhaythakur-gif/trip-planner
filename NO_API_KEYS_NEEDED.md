## ✨ How It Works

Your app automatically uses:

1. **Overpass API (OpenStreetMap)**

   - Same data that major map apps use
   - Unlimited requests
   - Community-maintained, always free

2. **Open-Meteo**
   - Professional weather forecasts
   - Used by thousands of apps
   - 10,000 requests/day free tier
   - No authentication required

3. **Nominatim (OSM)**
   - Geocoding service
   - Converts city names to coordinates
   - No authentication required

## 🎯 Data Quality

### Overpass API (OpenStreetMap)
- ✅ **Excellent coverage** - used by Apple Maps, Microsoft Bing, and many others
- ✅ **Constantly updated** by millions of contributors worldwide
- ✅ **Rich metadata** - opening hours, websites, phone numbers, Wikipedia links

### Why Skip OpenTripMap?
- Overpass gives you **unlimited** requests vs 1000/day
- Overpass has **better** worldwide coverage
- Overpass requires **zero** setup
- OpenTripMap is just another API layer over OpenStreetMap data
